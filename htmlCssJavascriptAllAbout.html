HTML, CSS and Javascript Cheat Sheet

HTML stands for HyperText Markup Language. It is the common language used to create web content and it describes the structure of a webpage. It consists of several elements that wraps or encloses different parts of content, giving instructions to the browser on how to display content.

Tags are used to create elements. Enclosing tags can be used in a variety of ways: to create a word or image hyperlink to a different page, to edit font styles or font sizes and to change the design of the page. This can all be done within the HTML file or by adding a reference link to a CSS page. CSS stands for Cascading Style Sheets and it refers to the language used to present and style a webpage.

Javascript is a programming language that can update and change HTML and CSS. It is commonly used by web developers to create interactions within web pages, software applications servers and games. CSS alone doesn't have the capability to format HTML elements. However, Javascript can work seamlessly alongside HTML and CSS to allow this.
